# sh-tovim
tovim mediate by pipe.

## How to Install
Example for zplug.
```zsh
zplug 'aiya000/sh-tovim', as:command, use:tovim
```

## Usage
Example for using.
```console
$ ls | tovim | xargs rm
```

You can choice rm target.
